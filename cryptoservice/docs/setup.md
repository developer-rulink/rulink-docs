# Настройка компьютера для работы с электронной подписью

Для работы с электронной подписью в сервисе необходимо настроить компьютер

1. Установить криптопровайдер - программу, которая работает с электронной подписью. [https://www.cryptopro.ru/products/csp/downloads](https://www.cryptopro.ru/products/csp/downloads)  
2. Установить плагиин - программа, которая позволяет передать данные из браузера к криптопровайдеру [https://cryptopro.ru/products/cades/plugin](https://cryptopro.ru/products/cades/plugin)
