# Сервис работы с электронной подписью

Добро пожаловать в сервис работы с электронной подписью.
Мы предлагаем надёжное решение для создания и проверки электронных подписей документов.

## Функционал сервиса

- [Проверка подписи](verification.md)
- [Редактирование подписи](editsigner.md)
- [Подписание документов](signpackage.md)

## Как это работает

Сервис использует открепленную электронную подпись. Это значит, что подпись создаётся отдельно от документа и сохраняется в отдельном файле.

Сервис доступен как через веб-интерфейс, так и через API. В данной документации вы найдёте подробные инструкции по обоим способам работы.

## Интеграция по API

Все наши сервисы разработаны с подходом API-First и доступны для интеграции.
Описание API: [https://rulink.io/api/v1/crypto/swagger](https://rulink.io/api/v1/crypto/swagger)
