# Набор инструментов для работы с PDF

## Функционал сервиса

Инструменты для работы с PDF. Сервис содержит более 80 инструментов для работы с PDF файлами.

## Как это работает

Вся обработка происходит в вашем браузере, это значит, что документы остаются на вашем компьютере и не передаются на сайт.

При выборе инструмента с сайта в ваш браузер загружается небольшая программа, которая обрабатывает pdf-файл.  
У этого сервиса нет API, сервис доступен только в веб-интерфейсе.

Каждый инструмент содержит полное описание и инструкцию использования.
